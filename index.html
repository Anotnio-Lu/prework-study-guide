HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The header of a web page in HTML can be created using the header tag, which is used to contain introductory content or a set of navigational links.
  • The footer of a web page in HTML can be created using the footer tag, which is used to contain information about the author, copyright information, or other related content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Control the layout of a web page, including the placement of elements, size, and position.
  • Set the color, font, and size of text, as well as the background color or image.
  • Apply effects such as drop shadows, borders, and transformations to elements.
  • Control the spacing and padding between elements.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Store and track changes to your code over time, including both the content and metadata of the changes.
  • Collaborate with others on a project by sharing changes between repositories.
  • Create branches to work on separate features or fixes in isolation from the main branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Create and manipulate arrays and objects to store and manipulate data.
